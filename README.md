### Installation [development]

- `cp config/example.config.json config/config.json` 
- `cp .example-env .env`
- `npm install`
- `npm install -g sequelize-cli mysql2`
- `npm run db-seeder-all` OR `npm run db-fresh`
- `npm run dev`
